symLPExperiments¶

symLPExperiments¶

The script, which is called to run the given examples of Lifted Linear Programming. This module retrieves LPs from specified files and provides the necessary interface to solve given LPs. Calculation and Computation is done in the C/C++ core, where most of the work is done.
