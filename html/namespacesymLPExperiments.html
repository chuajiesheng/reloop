Detailed Description

Documentation for this module.

This module retrieves LPs from specified files and provides the necessary interface to solve given LPs. Calculation and Computation is done in the C/C++ core, where most of the work is done.

def symLPExperiments.closeLP ( void  )

Closes the given LP by deallocating memory.

Closes LP by calling the destructor of glpk, which deallocates all memory used by the corresponding problem object iff the constructor was called in openLP
The call of glp_free_env() frees all resources used by GLPK routines
