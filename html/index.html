Saucywrapper

Introduction

Welcome to the documentation of saucywrapper. Below this short introduction you will find a tutorial on installing the necessary packages to run the program. Saucywrapper is a tool, which utilizes the symmetry discoverer Saucy, which is written in plain C.

Installation

The framework for the script is written in Python, but utilizes cython to call C++ functions for its computationally intensive operations.

Step 1: Installing Cython

First of all, to compile saucywrapper, you will need to install cython, which creates a shared library from specified C++ and .pyx files available for use in the python runtime environment. The documentation on cython and on how to install it can be found here. For more information about Cython and how to write Python/C hybrid code within the .pyx file domain please visit the here of Cython.

Step 2: Installing SciPy

Scipy is a package, which lets us represent arrays and matrices in python in a more efficient way. New data structures like coordinate and sparse matrices are made available to allow more scalability in the python code itself. If you are using Ubuntu,Debian or similiar distributions of said OS and have the necessary permissions you can simply execute

sudo apt-get install python-numpy python-scipy python-matplotlib ipython ipython-notebook python-pandas python-sympy python-nose+

Otherwise please see the more detailed description of the installation process here

Step 3 : Installing GLPK-Sol

The GNU Linear Programming Kit (GLPK) is used to [lift?] calculate the bounds and the objective of a given problem and then solve it by calling a standard linear solver for that matter. A detailed description on how to set-up GLPK for your OS can be found here

Step 4 : Installing zLib

To install the zLib programming kit simply run

$sudo apt-get install zlib1g-dev

Step 5 : Compiling the Code

At this point you should be able to compile an executable version of the code. This can be done by cloning into the repository and changing into the cloned directory :

$[clone saucywrapper link]
$cd saucywrapper

Once you have navigated to the folder where the code is (usually directly in the cloned directory) you can run :

$python wrapper-setup.py install

The shared library is now being compiled and should do so without any problems. If you get an error saying the your permissions were denied try to run either :

$python wrapper-setup.py install --user

or

$sudo python wrapper-setup.py install

if you have the necessary permissions to execute the sudo command.

Step 6 : Running the script

To test if the compilation and installation of all packages was successful you can run

$ python symLPExperiments.py

to execute the script with a given set of sample data, which is located in the /data folder. If you have seperate data to test feel free to edit the path to the LPs at the bottom of symLPExperiments.py. The resulting code of one file should be looking somewhat like this
